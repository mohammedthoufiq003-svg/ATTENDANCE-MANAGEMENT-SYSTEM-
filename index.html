Project Overview

The Attendance Management System is a web-based application designed to manage student attendance efficiently. It helps faculty record attendance, generate reports, and monitor student attendance percentages accurately.

Project Features

  • Student Registration
  • Faculty Registration
  • Subject Management
  • Attendance Marking
  • Attendance History
  • Attendance Reports
  • Attendance Percentage
  • Profile Management
