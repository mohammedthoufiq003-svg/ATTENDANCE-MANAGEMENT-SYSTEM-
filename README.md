# Attendance-Management-System

Attendance Management System is a web-based application designed to automate and simplify the process of recording and managing student attendance. The system enables administrators and faculty members to maintain accurate attendance records, monitor student attendance, generate reports, and improve overall academic administration.

## Attendance Management System

Attendance Management System is an intelligent web application developed to reduce manual paperwork and improve the efficiency of attendance management in educational institutions. The system provides secure login, student management, attendance tracking, report generation, and attendance analytics through a user-friendly interface.

## Features

- User Authentication (Admin & Faculty Login)
- Student Registration and Management
- Faculty Management
- Subject Management
- Daily Attendance Marking
- Attendance History
- Attendance Percentage Calculation
- Search Student Records
- Monthly Attendance Reports
- Downloadable Attendance Reports
- Dashboard with Attendance Statistics

## How to Run

### Clone Repository

```bash
git clone https://github.com/your-username/Attendance-Management-System.git
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

or

```bash
flask run
```
